# Clase-IMPLE-BD-4toG
Proyecto “Juego de la Ignorancia” realizado por Francia Naily Ortiz Contreras y Katia Minerva Mendez Tepec. Desarrollado en Python, Tkinter y MySQL. El juego consiste en una carrera de preguntas y respuestas donde los jugadores avanzan según sus aciertos. Incluye interfaz gráfica, base de datos, sistema de puntos, animaciones y ventanas dinámicas.
